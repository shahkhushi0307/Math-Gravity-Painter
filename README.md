# 🚀 Math Gravity Painter

## 🌌 About The Game

Math Gravity Painter is a physics-inspired educational puzzle game where players guide a glowing Gravity Ball by drawing paths, curves, and shapes. The game combines mathematics, creativity, logical thinking, and problem-solving in a fun space-themed adventure.

Built for students of Grades 5–9, the game transforms learning into gameplay instead of traditional question-answer methods.

## 🌍 Worlds

### 🌌 Fraction Galaxy
Learn fractions using fraction pieces, gates, circles, and matching puzzles.

### ➕ Integer Space
Explore positive and negative numbers using gravity zones and integer pathways.

### 📐 Geometry Planet
Draw shapes, create bridges, and solve area and perimeter challenges.

### 🧩 Algebra Station
Unlock doors, solve equations, and complete algebra-based puzzles.

### 🎲 Probability Zone
Use dice, spinners, and prediction mechanics to understand probability.
